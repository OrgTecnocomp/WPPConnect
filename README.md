# WPPConnect Tecnocomp

Build revisada e configurada do WPPConnect para uso da Tecnocomp Tecnologia e Serviços LTDA

## Estrutura

- Serviços
    - WPPC-FRONT
        - Interface WEB da aplicação

    - WPPC-SERVER
        - Serviço principal da aplicação

    - WPPC-NGINX
        - Proxy de Serviço da aplicação
- Redes
    - WPPC-NETWORK (Bridge)
